Principais formatações

Negrito / destaque

Nesta frase temos um termo em negrito, usando a tag <b> (não semantico)

Nesta frase temos um termo destacado usando o termo <Strong>(semantico)

Italico / Ênfase

Nesta frase vamos usar um terno em itálico usando a tag <i> (não semantico)

Nesta frase vamos usar em Ênfase , usando a tag <em> (semantico)

Texto marcado

Podemos utilizar também texto marcado, usando a tag <mark>

E no outro paragrado temos outro texto marcado no final.

Texto grande e pequeno

Estamos criando um texto grande, e pequeno nessa frase .

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto com inserido para dar uma ênfase e indicar que ele foi adicionado depois.
